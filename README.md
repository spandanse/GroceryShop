# GroceryShop (Java Project)

This is a simple Java program that simulates a grocery shop system.  
It demonstrates the basics of Java programming, including classes, objects, methods, and user input.

## 🚀 How to Run

1. Compile the program:
   javac GroceryShop.java

2. Run the program:
   java GroceryShop

✨ Features

User can view and manage grocery items.

Demonstrates Java basics (loops, conditions, arrays/objects).

Beginner-friendly project.

🛠️ Requirements

Java JDK (version 17 or later)
